# t06: float arrays

Write a Lumen program with:

- allocate a Float array of length 3 with `array(3)`
- set index 0 to `1.5`, index 1 to `2.5`, index 2 to `3.0` with `aset`
- sum all elements with a `while` loop bounded by `alen`, reading each with `aget`
- a `main(console: Console) -> Unit` that prints `round(sum * 100.0)` with
  `console.print_int` (the sum scaled by 100, as an Int)

Print nothing else.
